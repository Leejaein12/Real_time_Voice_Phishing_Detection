# Voice AI Project

보이스피싱 실시간 탐지 시스템

# 🎙️ Real-Time Voice Phishing Detection (Korean)

한국어 음성을 기반으로 **딥페이크(TTS) vs 실제 음성**을 구분하는 모델을 학습하고,  
보이스피싱 탐지에 적용하는 프로젝트입니다.

---

## 🚀 Overview

이 프로젝트는 Raw waveform 기반 모델을 활용하여:

- ✅ 실제 음성 (genuine)
- ❌ 합성 음성 (TTS, Deepfake)

을 구분하는 것을 목표로 합니다.

특히 한국어 환경에서의 **보이스피싱 탐지**를 고려하여 설계되었습니다.
